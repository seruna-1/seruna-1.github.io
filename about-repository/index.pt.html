Sobre esse repositório

Essa página está dentro de um diretório pessoal que contém uma gama (bem pequena, no momento) de obras, pensamentos, experiências e conhecimentos que eu gostaria de armazenar e tornar públicos.

Esse domínio inteiro é um repositório git licensiado sobre a licensa MIT. Isso significa que qualquer pessoa tem permissão para distribuí-lo e modificá-lo. Você pode acessar minha conta no Codeberg ou no GitHub, fazer uma cópia desse repositório e usar isso como base para estudar desenvolvimento web ou até usar isso como inspiração para construir seu próprio domínio, se isso lhe for útil.

Expansão de domínio!

As páginas daqui não se comunicam com um servidor. Quando você acessa uma página (essa, por exemplo) ela é despachada até o seu navegador a partir do servidor do Codeberg ou do GitHub. Eu mantenho uma cópia local desse domínio, a qual eu insiro nesses servidores, mas não mantenho um servidor próprio para o meu domínio. Por isso, não posso usar uma linguagem de programação (como Python) para manipular o conteúdo servido ao usuário.

O estilo das páginas é determinado pelo simple.css , cujo código está contido no diretŕoio [simple.css-2.3.1]. O diretório [hanabi] serve para estilo adicional que eu queira desenvolver.

Cada página está contida em um subdiretório no domínio (com exceção da página [index.html] no topo do domínio), e é formada por um arquivo html, um ou mais arquivos de css (que determinam o estilo) e um ou mais arquivos de javascrip (que adicionam funcionalidade).

Conteúdo presente em várias páginas pode ser manipulado por um utilitário que eu estou desenvolvendo. Ele está contido em [page-manipulation], mas talvez se torne um repositório separado.
