Vamos começar definindo multiplicação

Aqui, vale destacar que [A] é uma quantidade, [+A] é somar a quantidade [A] à uma quantidade já existente. Pelo mesmo raciocínio, [-A] é subtrair a quantidade [A] de uma quantidade anterior. Para simplificar a escrita, vamos considerar que, na notação [A], está implicita a operação de soma.

Então, como na imagem acima, multiplicar [B] por [A] significa repetir [+A] por [B] vezes. Em outras palavras, multiplicação é uma repetição da soma de uma mesma quantidade.
